# Tutoriel Forth 

Ce projet est un tutoriel pour le langage de programmation Forth. Je l'ai créé pour me pratiquer à écrire des programmes en [gforth](https://www.complang.tuwien.ac.at/forth/gforth/Docs-html/). 

Je travaille sur un PC avec Ubuntu 20.04 LTS et gforth version 0.7.3 

Le dossier racine contient des fichiers forth  librairies pour les autres programmes.

Chaque sous-dossier contient un programme écris en Forth. 

* Le dossier **premiers-pas** contient une variété de petits programmes.

* Le dossier **snake** contient un jeu en mode console texte.

* Le dossier **tiny-BASIC** contient un interpréteur tiny BASIC ainsi que des programmes BASIC pour le tester.
